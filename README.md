# andrewzamp.github.io

A minimal and clean research profile website template.

## Structure

- `index.html` - Main page with research profile layout
- `style.css` - Minimal, professional styling
- `_config.yml` - Jekyll configuration

## Sections

The template includes the following sections:
- **About** - Brief introduction
- **Research** - Research areas and interests
- **Publications** - List of publications
- **Contact** - Contact information and links

## Customization

Simply edit `index.html` to add your own content:
1. Replace "Your Name" with your actual name
2. Update the subtitle with your position and institution
3. Fill in each section with your information
4. Update contact links with your actual profiles

The design is minimal and responsive, working well on all devices.
